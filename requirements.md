# Software Requirements

## Functional Requirements 
1. User can create an account  
2. User can login to the system
3. User can update profile information
4. User can search for products
5. User can place an order
6. User can make online payments
7. User can securely login to the system

## Non-Functional Requirements
1. System should load within 2 seconds
2. System should be available 99.9% of the time
3. Data should be encrypted for security
4. System should support up to 1000 users simultaneously
5. Interface should be user-friendly
